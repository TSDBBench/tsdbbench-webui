Welcome to TSDBBench Web User Interface

TSDBBench UI provides a single point of communication with Elastic Infrastructure and Virtual Machine instances.

It's goal is to simplify the process of running TSDB benchmarks by automating VM instance creation, SSH keys generation and control over created VMs.

It was built to support multiple Cloud providers.
